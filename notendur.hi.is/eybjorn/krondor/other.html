There are many possible ways of classifying these items. The method chosen here may not be the best one, but should be easily understood. Many items are easily classifiable, but certain overlaps make a simple classifiaction of all these items near impossible.

I have chosen to group the items into four main classes: Items affecting Skills, Spell-related Items (Non-Combat), Spell-related Items (Combat), and Quest Items. Many items belong to more than one of these classes, but I have avoided any duplications. For example, an item which affects Skills, and is also a Quest item, will only be listed in the first class. In order to clarify, I have also used the following four codes (Red if true, Grey if false):

Inventory: Only needs to be in Inventory to be effective.
Use: Must be Used in order to be effective.
Quest: Necessary in order to complete a Quest.
Essential: The game can not be finished without this item.

In the fourth class (Quest Items) only the Essential code applies, so the others have been left out.
