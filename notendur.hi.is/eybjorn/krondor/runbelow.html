ZONE C1 - CAVALL RUN
Chapters 1, 2, 3 and 6

[ZONE 4 MAP] [CHAPTER 3]

This is an Image Map. Click Green Caches, Yellow Combats, or Red Arrows.


Cavall Run can not be entered unless you have the Knight's Piece found in the Kenting Rush Well. Once exited, it will not stay open, so make sure you don't lose the Knight's Piece, if you want to re-enter.

There are no enemies here in Chapters 1 and 2. All enemies appear in Chapter 3 (and will be carried over to Chapter 6, if not killed).

Chapter 3 will end, when you open the DARKNESS Chest.

DARKNESS CHEST
In order to access this Chest you will need Navon's CELLAR KEY, without which the Room can't be opened. This will not be possible until Chapter 3. The contents listed above are in the game code, but I don't think they can be accessible. Opening the DARKNESS Chest in Chapter 3 will only result in a dialogue which signals the End of the Chapter.

If Gorath and Owyn are thus inclined in Chapter 6, they will find different items in the DARKNESS Chest (see above). [Technically, you can access this version of the DARKNESS Chest in Chapters 1 and 2, if you have a CELLAR KEY. The game doesn't allow this, of course, but if you're inclined to cheat ...]
