NOTES ON LOCKS AND KEYS

There are three types of Locks:

Can only be opened with a Picklock
Can be opened either with a Picklock or a Key
Can only be opened with a Special Key

Each Lock has a Lock Number (LN). The higher the LN, the harder it is to open. Your success will depend on either your Lockpicking Skill (LPS), or your possession of the appropriate Key.

Any Lock with a LN of less than 50 can only be picked. Your LPS must be greater than the LN for this to be successful.

Each Key has a corresponding Key Number (KN). If the KN corresponds EXACTLY with a LN, the Lock will open. It will also open with Picklocks, if the LPS is higher than the LN. Thus:

LN/KN 50 = Peasant's Key or LP=51 (or higher)
LN/KN 60 = Guilder's Passkey or LP=61 (or higher)
LN/KN 70 = Noble's Passkey or LP=71 (or higher)
LN/KN 80 = Guildis Thorn or LP=81 (or higher)
LN/KN 90 = Virtue Key or LP=91 (or higher)

Any Lock with LN 51-59, 61-69, etc., can only be picked.

Some Locks are special, can NOT be picked, and need a Special Key to be opened. All, except one, of these Keys are unique:

LN/KN 101 = Key of Lineages
LN/KN 102 = Cellar Key (unique)
LN/KN 103 = Nivek's Key (unique)
LN/KN 104 = Interdictor Key (unique)
LN/KN 105 = Ward of Ralen-Sheb (unique)
LN/KN 106 = Royal Key of Krondor (unique)
