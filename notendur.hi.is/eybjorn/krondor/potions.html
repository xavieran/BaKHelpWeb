THE DRAGONTAIL WELL BUG(S).

Near Elvandar there is a Magical Well. By feeding it with Fadamor's Formula you can (supposedly) raise your Characters' permanent Strength by 3 points for each charge of Fadamor's. This feature is bugged in both versions of the game.

Version 1.01
Original diskette version
This works, more or less as promised. You can raise Strength by 3 points in exchange for each charge of Fadamor's. But the game can cheat you! I've tried to analyse this, and here's what I think happens: First Owyn's inventory is checked for Fadamor's Formula. If he has any, it will all disappear, not just one charge! If Owyn doesn't have any, Gorath's supply is checked. If he has any, it will all go, just like in Owyn's case! Are these guys clumsy, or what? There seems to be an obvious method to avoid this. First make sure Gorath is holding ALL the available Fadamor's. Now give ONE charge to Owyn. Use the Well. Repeat as often as you want. (If you don't want to take any chances on this, just place all the Fadamor's you have in a Bag, and then feed one charge at a time to either of the Characters.)

Version 1.02
Later diskette version = CD-ROM version = Free download
Probably the programmers became aware of this problem, and tried to correct it in the new version. They really bungled it. Now the Well doesn't consume ANY Fadamor's AT ALL. Even if you only have ONE portion, you can go on raising Strength for ages! Surely this was not meant to be. You can raise Strength to 250 this way (which is utterly silly), but beware! I would suggest stopping at 200 - any more Strength is useless, and practically dangerous. Why?

Here's the reason why. I don't want to go into this in hexadecimal detail, but 255 is the highest Strength figure the game can handle. If Strength goes above that number, it will revert to 0 (Zero). You don't want that to happen, do you? At this stage of the game, there are at least two possibilities left for this to happen. So don't be TOO greedy, please?
