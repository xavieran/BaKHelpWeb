Note on Moredhel Spellcaster 2.24.079/3-3:

In version 1.01 this enemy is bugged. You can't search his body after he's been killed. If you know how to use a hex-editor you can easily fix this bug.

Open the file startup.gam (this is the "save" the game loads when you start a new game). This file is in the dynamix\krondor directory.

The error is at address 294271 (hex 47D7F). Just change "07" to "04". This will effectively get rid of the bug.

This will, of course, not affect any saves you've already made. If you're already into the game, and don't want to start from scratch, you'll have to patch a saved-game file in the same way.
