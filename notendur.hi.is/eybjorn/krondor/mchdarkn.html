DARKNESS CHEST
Quest Chest (Chapter 3)
Cavall Run

[BACK TO RIDDLES]

The room containing this Chest can only be opened with the Cellar Key, which can only be obtained by killing Navon in Chapter 3. Opening the Chest results in the discovery of the Moredhel Attack Plans, and the end of this Chapter. The game code shows a list of items in this Chest. These items can hardly be accessed at any stage of the game (without serious hacking), but are shown at the bottom of this page.

The Chest can be re-opened in Chapter 6, if Gorath and Owyn are thus inclined. It will then contain different items (see below). [These items can be made available in Chapters 1-2 as well, if you don't mind cheating. All you need is the Cellar Key.]
